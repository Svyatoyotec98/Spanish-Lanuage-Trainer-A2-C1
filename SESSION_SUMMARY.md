| Группа | Количество слов |

| FAMILIA | 6 |
| IDIOMAS_APRENDIZAJE | 18 |

**Структура данных:**
```json
{
  "spanish": "el abuelo",
  "ru": "дедушка",
  "hardSentences": ["Mi ___ paterno tiene 85 años.", "El ___ de mi esposa cocina muy bien."],
  "hardSentencesRu": ["Моему дедушке по отцовской линии 85 лет.", "Дедушка моей жены очень хорошо готовит."]
```

## Важные функции в app.js

| Функция | Описание |
|---------|----------|
| `renderGroupCards()` | Рендер карточек групп в Palabras |
| `showMiniDictionary()` | Показ мини-словаря группы |
| `showGroupPreview()` | Промежуточный экран с кнопками |
| `renderGramaticaExercises()` | Рендер упражнений в Ejercicios |
| `renderGrammarList()` | Рендер списка в справочнике грамматики |
| `hideAll()` / `hideAllScreens()` | Скрытие всех экранов |
